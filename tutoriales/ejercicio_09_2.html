Teoría Básica

Conceptos Clave

En Python, puedes trabajar con archivos utilizando las funciones integradas open(), read(), write(), y close(). Los archivos se pueden abrir en diferentes modos: lectura ('r'), escritura ('w'), y adición ('a').

Es importante cerrar los archivos después de usarlos para liberar recursos del sistema.

Archivo de Ejemplo

Archivo: mi_diario.txt

Este archivo contendrá las entradas del diario personal.

Fecha: 2023-10-01

Hoy he aprendido a manejar archivos en Python. ¡Es fascinante!

Fecha: 2023-10-02

He escrito mi primera entrada en el diario digital. ¡Me encanta!

Código de Ejemplo

# Abrir el archivo en modo escritura
diario = open('mi_diario.txt', 'w')
# Escribir una entrada en el diario
diario.write('Fecha: 2023-10-01\\nHoy he aprendido a manejar archivos en Python. ¡Es fascinante!\\n')
# Cerrar el archivo
diario.close()
# Abrir el archivo en modo lectura
diario = open('mi_diario.txt', 'r')
# Leer el contenido del diario
contenido = diario.read()
# Mostrar el contenido
print(contenido)
# Cerrar el archivo
diario.close()

Modos de Archivo

Modo Lectura (r)

Abre un archivo para leer. Si el archivo no existe, se genera un error.

Modo Escritura (w)

Crea un nuevo archivo o sobrescribe uno existente. El contenido previo se pierde.

Modo Adición (a)

Añade contenido al final del archivo sin borrar el contenido existente.

Pasos para Crear y Leer el Diario

Abrir el archivo en modo escritura para crear o sobrescribir.
Escribir la entrada del diario con la fecha y el contenido.
Cerrar el archivo para guardar los cambios.
Abrir el archivo en modo lectura para leer las entradas.
Leer el contenido del archivo y mostrarlo en la consola.
Cerrar el archivo después de leer para liberar recursos.

Comparación de Modos de Archivo

Modo Descripción Permisos
r Lectura del archivo existente. Sólo lectura.
w Escritura, crea o sobrescribe el archivo. Lectura y escritura.
a Añade contenido al final del archivo. Lectura y escritura.

Consejo

Siempre verifica si el archivo existe antes de abrirlo en modo lectura para evitar errores.

Advertencia

Usar el modo escritura ('w') sobrescribirá el contenido del archivo. Asegúrate de que es lo que deseas.

Ejercicio Propuesto

Escribe un programa que permita al usuario agregar entradas a su diario personal. El programa debe:

  • Pedir al usuario la fecha y el contenido de la entrada.
  • Guardar la entrada en un archivo llamado mi_diario.txt.
  • Permitir al usuario leer todas las entradas guardadas.

Utiliza excepciones para manejar posibles errores al abrir o escribir en el archivo.

Ejemplo de Código

try:
    diario = open('mi_diario.txt', 'a')
    fecha = input('Ingrese la fecha (YYYY-MM-DD): ')
    contenido = input('Ingrese el contenido de la entrada: ')
    diario.write('Fecha: ' + fecha + \\n')
    diario.close()
    diario = open('mi_diario.txt', 'r')
    print('Contenido del diario:')
    print(diario.read())
    diario.close()
except Exception as e:
    print('Error al manejar el archivo:', e)
