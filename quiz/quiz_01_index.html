Preparación para el Cuestionario de Python

Instrucciones del Cuestionario

  • El cuestionario consta de 30 preguntas de opción múltiple
  • Tienes que seleccionar una sola respuesta por pregunta
  • Cada pregunta tiene un valor de 1 punto
  • No hay penalización por respuestas incorrectas
  • Al finalizar obtendrás tu puntaje y retroalimentación
  • No hay límite de tiempo

Conceptos Clave a Revisar

Sintaxis Básica

  • Comentarios con #
  • Función print() para mostrar información
  • Función input() para recibir datos del usuario
  • Indentación (sangría) para bloques de código

Tipos de Datos

  • Enteros (int) y decimales (float)
  • Cadenas de texto (str)
  • Booleanos (bool: True y False)
  • Conversión entre tipos con int(), str(), etc.

Operadores

  • Aritméticos: +, -, *, /, // (división entera), % (módulo)
  • Comparación: ==, !=, <, >, <=, >=
  • Lógicos: and, or, not

Estructuras de Control

  • Condicionales: if, elif, else
  • Bucles: while y for
  • Palabras clave: break, continue, pass

Estructuras de Datos

  • Listas: [], métodos como append(), remove()
  • Tuplas: () (inmutables)
  • Diccionarios: {clave: valor}
  • Conjuntos: {} (sin elementos repetidos)

Funciones

  • Definición con def nombre_funcion():
  • Parámetros y argumentos
  • Valores de retorno con return
  • Ámbito de variables (scope)

Recomendaciones

  • Lee cuidadosamente cada pregunta
  • Si no estás seguro, marca la opción que consideres más probable
  • Administra bien tu tiempo
  • Las preguntas pueden tener más de una respuesta aparentemente correcta
  • Revisa tus respuestas antes de finalizar
